# libtiff

TIFF image library packaged for the Zig build system.

## About

This is [libtiff](https://libtiff.gitlab.io) packaged for the Zig build system.

## Dependencies

- zig-build-helper
- libjpeg-turbo (optional)
- zlib (optional)
